# 区域作成ストーリー

1. 管理者は領域を新規作成する
2. 編集者は領域に区域親番を追加する。

- 仕様書上は`001`などの通し番号を想定しているが、英数字を許容する。（仕様変更）
- 領域内一意とする

3. 編集者は区域親番に区域を追加する。

- 仕様書上は`05`などの通し番号を想定しているが、英数字を許容する。
- 区域親番内一意とする

4. 編集者は区域から「ポリゴン作成」を開始する。
5. 編集者は地図上でポリゴンを作成する。
6. 編集者は区域親番とそれらに含まれる区域およびポリゴンの作成を完了する。
7. 編集者は管理者に区域一式の作成完了を報告し、承認申請する。
8. 管理者は区域一式を承認する。
9. 承認された区域がLinkSelfグループ内で区域として運用される。
