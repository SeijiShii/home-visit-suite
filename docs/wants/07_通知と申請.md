# 通知と申請

## 通知

- 任命招待
  - 招待を受けた人が視認できる箇所に表示する
- 区域割り当て
  - 割り当てを受けた人のマイページに表示される
  - 返却・回収された区域は「返却済み」として一定期間（1か月）表示する
- 申請
  - 編集スタッフのタスクリストに表示される
  - 申請結果が受理されたかどうかは申請者は関知しない（データ反映をもって受理とする）

## 申請内容の取扱い

- 活動スタッフからの申請内容を区域に反映する
  - 区域情報の更新
  - 訪問をしない家や場所
    - タスクとして管理する
      - 未処理、保留、処理済み
- 「訪問しない家」の期限管理
  - システム側で期限の仕組みを定義しない
  - 運用者側で期間や判定基準を持つ

## 監査ログ

- 権限変更・区域編集・訪問不可設定などの重要操作の履歴を記録する
- 管理者・編集スタッフが閲覧できる

## データ保持・アーカイブ

- LinkSelfの設定値として保持期間を管理者が設定できる
- 管理者はLinkSelfデータをダンプできる
