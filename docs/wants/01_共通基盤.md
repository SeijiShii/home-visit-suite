# 共通基盤

## データ構造方針

- 全てのデータは木構造とする
- データの競合解決（同期・マージ）はLinkSelfのカバー範囲とする

## 用語定義

- 本仕様における「区域」は運用上の最小単位（枝番レベル）を指す
  - 正確には区域とは枝番の集合だが、メンバーが主に扱うのは枝番レベルであるため「区域」と呼称する
- 枝番の集合としての上位概念は「区域親番」と呼ぶ

## アプリの関係

- 管理メンバー向けアプリは活動メンバー向けアプリの上位互換である
  - 管理アプリから訪問記録の閲覧・編集も可能
  - 管理アプリの地図は編集可能、活動アプリの地図は閲覧・訪問記録用

## 実行環境

- 管理メンバー向けアプリ
  - Windowsアプリ
  - Macアプリ
- 活動メンバー向けアプリ
  - モバイルネイティブ（iOS/Android）を想定
  - LinkSelfはサーバーレスのためWebは想定しない

## データインフラ

- LinkSelfを使用
  - https://github.com/SeijiShii/link-self
  - 完全サーバーレス
    - データはLinkSelfグループ外に出ることはない
- 管理メンバーと訪問活動メンバーは同じグループに所属し、ロールが異なる

## マイページ

- 各アプリはメンバーのマイページを持ち、必要な情報や通知を一覧できる
- UIUXは追って検討する
