# 画面設計

## 画面一覧と優先度

| # | ルート | ページ | 状態 | 優先度 | 依存Binding |
|---|---|---|---|---|---|
| 1 | `/map` | 地図・ポリゴン編集 | **実装済み** | — | MapBinding, RegionBinding |
| 2 | `/regions` | 領域・区域管理 | **実装済み** | — | RegionBinding |
| 3 | `/activities` | 訪問活動管理 | プレースホルダー | **高** | ActivityBinding |
| 4 | `/users` | ユーザー・グループ管理 | プレースホルダー | **高** | UserBinding |
| 5 | `/` | ダッシュボード | プレースホルダー | 中 | 複合（通知+活動） |
| 6 | `/coverage` | 網羅管理 | プレースホルダー | 中 | CoverageBinding |
| 7 | `/requests` | 申請管理 | プレースホルダー | 低 | NotificationBinding |

## 3. 訪問活動管理 `/activities`

### 画面構成

```
┌─────────────────────────────────────────────────────┐
│ 訪問活動                                    [+ 貸出] │
├─────────────────────────────────────────────────────┤
│ [活動中] [返却済み] [完了]            [検索: ______] │
├──────────────┬──────────────────────────────────────┤
│ 活動一覧      │ 活動詳細                              │
│              │                                      │
│ ▸ NRT-001-01 │ 区域: NRT-001-01 加良部1丁目-01       │
│   活動中      │ 種別: 貸し出し                        │
│   田中太郎    │ 担当者: 田中太郎                      │
│              │ 貸出者: 鈴木編集                      │
│ ▸ NRT-001-02 │ ステータス: 活動中                     │
│   活動中      │ 開始: 2026-03-01                     │
│   鈴木次郎    │                                      │
│              │ ─── チーム割り当て ───                  │
│ ▸ NRT-002-01 │ 2026-03-25  チーム1 (田中,佐藤)       │
│   返却済み    │ 2026-03-20  チーム3 (伊藤,渡辺)       │
│   佐藤三郎    │             [+ チーム割り当て]         │
│              │                                      │
│              │ ─── 訪問記録 ───                       │
│              │ 12件 (会えた: 8, 留守: 4)              │
│              │                                      │
│              │ [返却] [強制回収]                      │
└──────────────┴──────────────────────────────────────┘
```

### 操作 → Binding対応

| 操作 | Service メソッド | ロール制限 |
|---|---|---|
| 貸し出し（+ ボタン） | `ActivityService.Checkout(lending)` | editor+ |
| 返却 | `ActivityService.Return` | 担当者 |
| 強制回収 | `ActivityService.ForceReturn` | editor+ |
| チーム割り当て | `ActivityService.AssignTeam` | editor+ |
| ステータスタブ切替 | `ActivityBinding.ListActivities(filter)` | all |
| 活動詳細表示 | `ActivityBinding.GetActivity` + `ListAssignments` + `ListVisitRecords` | all |

### 貸し出しダイアログ

```
┌─── 区域の貸し出し ────────────────────┐
│                                      │
│ 区域:     [区域選択ピッカー ▾]        │
│ 担当者:   [メンバー選択 ▾]           │
│                                      │
│       [キャンセル]  [貸し出し]        │
└──────────────────────────────────────┘
```

- 区域選択: 既存のAreaPickerDialogを流用。チェックアウト済みの区域はグレーアウト
- 担当者選択: ユーザー一覧から選択

## Wails Binding 一覧（新規必要分）

| Binding | メソッド | 対応Service/Repository |
|---|---|---|
| **ActivityBinding** | ListActivities, GetActivity, Checkout, Return, ForceReturn, ListAssignments, AssignTeam, ListVisitRecords | ActivityService + ActivityRepository |
| **UserBinding** | ListUsers, GetUser, SaveUser, DeleteUser, ListGroups, SaveGroup, DeleteGroup, ListTags, SaveTag, DeleteTag, ListInvitations, InviteToRole, AcceptInvitation, DismissRole, RemoveMember | AuthService + UserRepository |
| **CoverageBinding** | ListCoverages, GetCoverage, ListCoveragePlans, SaveCoveragePlan, ApprovePlan, ListAreaAvailabilities, SaveAreaAvailability | CoverageRepository |
| **NotificationBinding** | ListNotifications, MarkRead, ListRequests, SaveRequest, ListAuditLogs | NotificationRepository |

## 実装順序

| Phase | 画面 | 理由 |
|---|---|---|
| **A** | Activities + ActivityBinding | 訪問活動が業務の中核 |
| **B** | Users + UserBinding | ロール管理がActivitiesの権限に影響 |
| **C** | Dashboard | 通知+活動サマリー。A/Bの成果物を組合せ |
| **D** | Coverage + CoverageBinding | 網羅管理は活動が回り始めてから |
| **E** | Requests + NotificationBinding | 申請は活動スタッフアプリ連携後 |
